# Blog Post Design: horizOn February 2025 Massive Update

## Context
horizOn is a game backend-as-a-service (BaaS) for indie game developers. This blog post announces multiple major updates shipped in a short timeframe.

## Target
- **Audience:** Indie game developers (solo devs, small teams)
- **Language:** English
- **Tone:** Casual, direct, devlog-style ("we shipped X")
- **Length:** ~800-1200 words
- **SEO Focus:** High-volume generic keywords (game backend, game BaaS)

## Approach
"Massive Update Roundup" — authentic devlog style, all updates as one cohesive story showing momentum. Each H2 section targets its own SEO keywords.

## SEO Meta
- **Title:** `horizOn February 2025 Update: Unreal SDK, Crash Reports, MCP Server & More`
- **Meta Description:** `horizOn ships major updates: Unreal Engine SDK, crash reporting, AI-powered MCP server, mobile-friendly dashboard, global load balancing, and a free self-hosted game backend.`
- **Primary Keywords:** game backend, game BaaS, game backend as a service
- **Secondary Keywords:** unreal engine backend, crash reporting games, self-hosted game backend, MCP server games, indie game backend
- **URL slug:** /blog/february-2025-massive-update

## Structure

### H1: horizOn Just Shipped Its Biggest Update Yet — Here's Everything New
- Intro (2-3 sentences): What horizOn is + "we've been shipping non-stop"

### H2: Unreal Engine SDK — Full Engine Support
- ~100-150 words
- Now supports Unity + Godot + Unreal
- Why this matters (covers all major engines)
- CTA: Link to SDK

### H2: Crash Reporting — Know When Things Break
- ~100-150 words
- Automatic crash grouping
- Dashboard integration
- No extra tool needed

### H2: MCP Server — AI Meets Your Game Backend
- ~100-150 words
- Brief explanation what MCP is
- What you can do with it (AI assistant interacts with horizOn directly)
- Why devs should care

### H2: Dashboard Redesign — Cleaner, Faster, Mobile-Ready
- ~100-150 words
- Visual upgrade
- Mobile-friendly
- Better UX

### H2: Global Load Balancer — One Domain, Worldwide
- ~100-150 words
- No more regional subdomains
- Simpler integration
- Better performance

### H2: Simple Server — Self-Host Your Game Backend for Free
- ~100-150 words
- Open-source PHP, zero dependencies
- API-compatible with horizOn BaaS
- Perfect for prototyping or full control

### Outro
- 2-3 sentences recap
- CTA to horizon.pm
- "Try it, it's free"

## Pixel Art Image Prompt
**Concept:** Retro 16-bit/SNES-era pixel art "command center" showing a stylized game dev dashboard. Dark background (matching horizOn's dark theme). Elements include: an Unreal-style logo, a crash/bug symbol, a server rack, a globe with connection lines, and a mobile phone showing a dashboard. Glowing accent colors (blue, purple, green).

**Style:** 16-bit pixel art, dark background, neon accents, wide format (16:9 for blog header).

## Deliverables
1. Blog post in Markdown (~800-1200 words)
2. Pixel art image generation prompt
